# ⚙️ Curso Bootstrap 5

⚙️ Projeto de aprendizado de Bootstrap 5 com exemplos práticos de layout, grid, utilitários, componentes, ícones e design responsivo. Demonstra fundamentos do Bootstrap, design web responsivo e implementações de UI para desenvolvimento frontend.

---

## 📂 Estrutura do Projeto

- **layouts/** – exemplos de layout utilizando o sistema de grid do Bootstrap  
- **components/** – cards, navbars, modals, buttons e outros componentes  
- **utilities/** – exemplos de classes utilitárias (margens, paddings, cores, display)  
- **forms/** – formulários responsivos e estilizados  
- **icons/** – uso de Bootstrap Icons  
- **examples/** – pequenas páginas de demonstração integrando múltiplos elementos  

---

## 🛠️ Tecnologias

- HTML5  
- CSS3  
- Bootstrap 5  
- Design Web Responsivo (RWD)  
- Bootstrap Icons  

---

## 🚀 Funcionalidades

- Layouts totalmente responsivos usando o sistema de grid do Bootstrap  
- Componentes prontos: cards, navbars, botões, modals  
- Classes utilitárias para espaçamento, tipografia e display  
- Formulários estilizados e com validação  
- Uso de ícones do Bootstrap para UI moderna  
- Exercícios práticos para reforçar o aprendizado  

---

## 💻 Como Utilizar

1. Clone o repositório:

```bash
git clone https://github.com/Albertinesilva/curso-bootstrap_5.git
```

2. Abra os arquivos HTML no navegador:

```bash
cd curso-bootstrap_5
```

- Você pode abrir qualquer arquivo HTML diretamente no navegador ou usar um servidor local para testar.

3. Explore as pastas para ver layouts, componentes, utilitários e exemplos de páginas.

🎯 Objetivo

Esses projetos foram criados como parte de um curso de Bootstrap 5 para aprendizado prático. Demonstra o uso de componentes, layouts responsivos e classes utilitárias no desenvolvimento frontend. Ideal para iniciantes e desenvolvedores em nível intermediário reforçarem suas habilidades.
